CSS Styles

This is just a reminder of some of the CSS sectors that I learned in the past units in my Codecademy Front-end Engineer course.

Typography CSS

Selector Description Example
color Defines the color of the text color: #05ffb0;
font-family Selects the font that will be diplayed font-family: Arial;
font-weight Defines the weight of the text font-weight: bold;
font-style Defines how much the text is slanted font-style: italic;
font-size Defines the size of the text font-size: 20px;
